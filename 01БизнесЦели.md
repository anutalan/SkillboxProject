## 1. Бизнес цели 
*1. Увеличение продаж собственного спортивного товара и инвентаря*

- Использовать данные о тренировках и используемом инвентаре пользователя, чтобы предлагать схожие товары для обновления уже приобретенных или для рекламы того, что нужно в данном виде спорта, но пользователь не отметил, что такой инвентарь или одежда у него уже есть; проводить акции перед значимыми спортивными мероприятиями, проводимыми как для спортсменов, так и для любителей.

  - Собирать информацию о продажах, в том числе во время промо акций, по переходам в собственные приложения для продажи наших товаров
 

*2. Укрепление бренда в спортивной среде*

- Для пользователей должно стать привычным, что наш бренд является естественным выбором и он присутствует в разных видах спорта

  - Необходимо вести срез по количеству пользователей, которые приобретают товар нашего бренда

*3. Увеличение вовлеченности в спорт*

- Стимулировать регулярные тренировки, соревнуясь с другими пользователями и со своими прежними результатами.
- Стимулировать пользователей вести здоровый образ жизни, посещать спортивные мероприятия, искать друзей среди пользователей платформы, которые тренируются рядом, по схожим маршрутам

  - Собирать срезы данных с результатами активности пользователей, установить норму для выполнения бизнес цели для каждой категории пользователя (спортсмен\ любитель + ребенок\подросток\взрослый). 

  - Собирать информацию по переходам в собственные отдельные приложения отдельных видов впорта

  - Собирать показатели активностей пользователей в чатах, группах по интересам

*4. Создание платформы для любой точки мира*

- Построить платформу, которая легко разворачивается в разных облаках других стран

- Платформа-констуктор должна быть достаточно легко конфигурироваться для добавления новых видов спорта, создания соответствующих клубов, чатов, создания промо акций для нового спорта, настраивать виды используемого инвентаря

- Платформа должна позволять получать данные разных устройств-датчиков для отображения результатов в своем приложении

- Вести срез срока поставки на прод настройки нового вида спорта, срок поставки контента в приложении для промо акций, срок настройки нового устройства-датчика самим пользователем для синхронизации их данных
